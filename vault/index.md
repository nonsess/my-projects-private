# Каталог vault

Навигация по хранилищу. Агент не ведёт этот файл вручную — используй `find` или `grep` для поиска заметок.

## Структура

| Директория | Что хранится |
|---|---|
| `profile/` | Профиль владельца (правки только с «ок») |
| `project/` | Карточки проектов |
| `goals/` | Цели: today / week / month / quarter / year |
| `tasks/` | backlog / in_progress / blocked |
| `decisions/` | Одно решение — один файл (append-only) |
| `people/` | Карточки людей |
| `notes/` | Атомарные заметки по темам |
| `insights/` | Паттерны поведения владельца (append-only) |
