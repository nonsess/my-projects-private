# Изоком — интеграция MAX мессенджер

## MAX Bot API (2026)

- Эндпоинт: `https://platform-api.max.ru/v1` (старый `botapi.max.ru` устарел)
- Авторизация: заголовок `Authorization: token <bot_token>` (query-параметры больше не поддерживаются)
- Отправить сообщение: `POST /messages?chat_id={id}`, тело `{"text": "...", "format": "HTML"}`
- Rate limit: 30 rps
- SDK: Python, Go, TypeScript — на `dev.max.ru`

## Требования для регистрации бота

- Только верифицированные юрлица РФ (подтверждение через Госуслуги или банк)
- ИП/физлица/нерезиденты — не могут зарегистрировать

## Сервер вне РФ

- Outbound HTTP к `platform-api.max.ru`: работает с любого IP, geo-блокировки нет
- Webhook (для входящих): работает технически, требуется HTTPS 443 + CA-сертификат
- Юридически: требование о хранении данных в РФ, но технического блока нет
