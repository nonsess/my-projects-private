# Лид-магниты и захват заявок (ресёрч 2026)

Как сделать, чтобы лиды сами заходили и оставляли заявки. Применяется к [[restogood-landing]]. Связано с [[selling-landing-pages]]. Источники — в конце.

## Главный сдвиг 2026: интерактив бьёт статику

- Интерактивные магниты (калькулятор / квиз / мгновенный аудит) конвертят **в ~2.4× лучше** статичных PDF. 91% B2B-покупателей предпочитают интерактивный контент статике.
- Статичные ebook/«гайд на 50 страниц» — мертвы: 1–8% opt-in. У всех есть, ценности ноль.
- Работает то, что **узко, быстро потребляется и сразу полезно** вокруг одного ценного результата.

## Бенчмарки конверсии (opt-in)

| Тип трафика / формата | Конверсия |
|---|---|
| Тёплый трафик (уже знают тебя) | 40–80% |
| Холодный (реклама/органика) | 10–20% |
| Средняя по рынку (любой магнит) | ~4% |
| Интерактив (калькулятор/квиз/аудит) | 30–50% |
| Квиз (запуск квиза → лид) | ~40% (B2B-услуги ~38%) |
| Чек-лист (лендинг) | 20–35% |
| Бенчмарк-отчёт | 15–30% |
| Длинный ebook / вебинар без конкретики | 1–12% |

## Топ-форматы для B2B-услуг (что подходит RestoGood)

1. **ROI-калькулятор** — сильнейший для B2B. Превращает скептика в самоквалифицированный лид за <3 мин: заменяет твой питч цифрами, которые посчитал он сам («люди верят математике, которую сгенерировали сами»). ~45% привязка к выручке.
2. **Бесплатный аудит / разбор** — лучший для консалтинга и сервисных B2B. Пример из ресёрча: веб-агентство даёт бесплатный 30-мин разбор текущего сайта → притягивает ровно тех, кто в рынке. Привлекает горячих.
3. **Бенчмарк / оригинальное исследование** — «сколько в среднем теряет ресторан на комиссии» — 14–28% конверсии, плюс PR-эффект.
4. **Чек-лист** — узкий, быстрый («10 пунктов: готов ли ваш ресторан к прямым заказам»).

## Правила самого магнита

- Узкое обещание, мгновенная отдача. Не нужен гигантский ebook — нужна релевантность, скорость, ясность.
- **Минимум полей формы** — в идеале только email/контакт. Каждое лишнее поле = drop-off.
- CTA про выгоду, не про действие («Узнать, сколько вернёте», не «Отправить»).
- Problem-Promise структура лендинга магнита: назвать боль → магнит как путь к облегчению.

## Дистрибуция — чтобы лиды «сами заходили»

Магнит без трафика не работает. Планируй дистрибуцию ДО того, как строишь.

- **SEO / контент** под проблемные запросы — статьи ранжируются по «боли», магнит ловит конверсию. Прицел: запросы вокруг ухода от комиссии, прямых заказов, удержания гостей.
- **Onsite-захват в моменты интента**: exit-intent попап на bounce-страницах, sticky-bar, time-delayed подсказки. Таргет по поведению, не просто по странице. Конвертят сильнее статичных форм.
- **Соцсети / LinkedIn** — №1 для B2B: таргет по должности, индустрии, размеру компании. Для РФ — VK/Telegram-каналы рестораторов, профильные сообщества.
- **Рефералы** — для hospitality-tech дают **самые качественные лиды по самой низкой цене**. Один довольный ресторан → следующие.
- **Email-нуртуринг по сезонам** (ROI $36 на $1): триггеры под бюджетный сезон (Q4), подготовку к пику. Сегментация по типу/размеру заведения.
- **Отзывы на площадках** где рестораторы валидируют выбор — кейсы с конкретными ROI и сроками внедрения.

## Ниша: как притягивать рестораны (2026)

- Контент под циклы общепита: бюджет Q4, подготовка к сезону весной.
- Таргет ЛПР: владелец / управляющий / маркетолог сети.
- Партнёрства: консультанты по общепиту, поставщики смежного софта, интеграторы.
- Сезонные нуртур-цепочки (чек-лист подготовки к пику и т.п.).

## Применение к RestoGood (рычаги по приоритету)

1. **Калькулятор уже есть — но он НЕ магнит.** PDF-КП скачивается без захвата контакта. Главный быстрый рычаг: **поставить мягкий захват** (email / Telegram / телефон) на скачивание КП или на «Обсудить подключение». Калькулятор + PDF = готовый сильнейший лид-магнит, сейчас он «играется», но не собирает лиды.
2. **Добавить «Бесплатный разбор»** (free audit) под нишу: «пришлите ссылку на ваше текущее онлайн-меню / страницу — за 1 день покажем, сколько теряете на комиссии и что починить». Притягивает горячих, даёт повод для созвона = заявка.
3. **Onsite-захват**: exit-intent / sticky-bar с оффером калькулятора или разбора — ловить уходящих.
4. **Дистрибуция под старт** (совпадает с bottleneck проекта — нет трафика): SEO-статьи под проблемные запросы → ведут на калькулятор; прямой аутрич + рефералы от первого реального кейса.
5. Юр.ограничение: агрегаторы по имени в публичном контенте не называть (см. [[feedback_legal_aggregators]]).
