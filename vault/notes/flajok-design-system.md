# Флажок — дизайн-система лендинга

Сервис отслеживания снижения цен на квартиры (ЦИАН/Авито → письмо при падении цены). Стек: Next.js 14 App Router, Tailwind, shadcn/ui, шрифт Onest. Код: `realestate-tracker/frontend/src/app/page.tsx`.

Финальное направление (выбрано 2026-06-20): **минималистично, чётко, технологично, читаемо** + одна сильная фишка из мира отслеживания цен. Путь Linear / Vercel / Raycast: много воздуха, сдержанная типографика, идеальное выравнивание по сетке.

## Палитра

| Токен | Hex | Применение |
|---|---|---|
| Фон | `#FFFFFF` | основной |
| Фон-секции | `#FAFAFA` | чередование секций, карточки-блоки |
| Текст (ink) | `#111111` | заголовки, основной |
| Muted | `#6B7280` | body-текст |
| Faint | `#9CA3AF` | подписи, мета |
| Hairline | `#ECECEC` | все границы и разделители |
| Акцент (drop) | `#EF4444` | CTA, снижение цены, −% |
| Green (saved) | `#16A34A` | только сумма экономии |

**Один акцент** — красный = действие И снижение цены (семантически точно). Красным красятся только кнопки, −% и ключевые ссылки. Зелёный — исключительно для суммы экономии.

## Типографика

- Один шрифт **Onest**, веса 400–900. Никаких вторых шрифтов.
- Заголовки секций: `clamp(1.75rem, 3.2vw, 2.5rem)`, line-height 1.05, letter-spacing −0.03em, font-black.
- Hero h1: `clamp(2.5rem, 5.5vw, 4rem)`, letter-spacing −0.035em.
- Body: 15–16px, `#6B7280`, line-height relaxed.
- **Все цены и числа — `font-mono` + `tabular-nums`** → читаются как данные, не как текст. Это ядро «технологичности».
- Eyebrow-лейблы: короткая красная черта (`w-5 h-px bg-primary`) + mono 11px uppercase tracking-[0.2em] `#9CA3AF`.

## Сетка и ритм

- Контейнер `max-w-5xl mx-auto px-6`.
- Секции `py-20 lg:py-24`, разделены hairline-границей `#ECECEC`.
- Блок статистики — сетка `gap-px bg-[#ECECEC]` (тонкие линии-разделители вместо отступов).

## Сигнатура (фишка)

1. **Hero-карточка падения цены** — карточка-«письмо», где цена тикает вниз (count-down rAF 1.4с, easing `1−(1−t)^4`), затем оседает: старая цена зачёркивается, прилетает красный тег −% (scale-in), появляется зелёная сумма экономии. Внизу — мини-sparkline, идущий вниз (SVG path + градиентная заливка). Это момент ценности продукта в один взгляд.
2. **Дашборд «Снижения за сегодня»** — чистый список-таблица реальных снижений (адрес · м² · цена · −%), как терминал мониторинга.

## Анимации

- **Lenis** smooth scroll (`src/components/SmoothScroll.tsx`, duration 1.1, easing `1−(1−t)^4`). Подключён в layout.
- Scroll-reveal fade-up через IntersectionObserver, 0.6s, stagger по 70мс.
- Marquee-тикер снижений: 20s linear, edge-fade по краям.
- `prefers-reduced-motion` уважается.

## Антипаттерны (убрано — выглядело как «нейросайт»)

- Пульсирующие точки (live-dot, зелёные индикаторы) — главный признак нейрослопа.
- Лишние pill-бейджи и теги-метки.
- 3D-tilt карточек на мышь — выглядит несвязно.
- Тёмные блоки невпопад на светлом сайте (тёмный navbar с белым текстом на белом фоне = невидимый).
- Гигантская растрёпанная editorial-типографика вперемешку с карточками — «не в попад».
- Точки в конце заголовков.

## Грабли

- В dev-режиме Next.js CSS инжектится через JS → в headless-скриншоте отваливается (рендерится голый HTML с serif). Проверять дизайн только на `next build` + `next start`.
- `pkill` во время `next build` рвёт сборку (exit 144) → рассинхрон хешей → CSS отдаёт 500. Билд не прерывать.
- snap-chromium пишет `--screenshot` только в `$HOME`, не в `/tmp`.
